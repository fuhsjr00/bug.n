Launching

Instructions for launching bug.n can be found here Installation

Introduction

Using bug.n can be done in pure keyboard control or through some extra mouse handling. See the Default_hotkeys for what are the default bindings. They are used in this tutorial.

By default there are 9 tags. These appear at the left hand side of the title bar. Each monitor has its own set of tags

Window model

Launch a few folders using [Win]+[e] and bug.n will tile the windows between the master and stack. A new terminal appears on the master window. Existing windows are pushed upon a stack to the right of the screen. [Win]+[Cntrl]+[Enter] toggles windows between master and stack.

Moving Around

To move to the next window in the list of windows, press [Win]+[DownArrow]. To move to the window, before the current window, in the list of windows, press [Win]+[UpArrow]. The window numbering information can be obtained by using [Win]+[Shift]+[i]..

To focus on another tag, press [Win]+[tag number].

You can click tags with the left mouse button and simulating [Win]+[tag number], but you can also click another tag with the right mouse button in order to bring those windows additionally into your current focus.

To kill a window, press [Win]+[c].

Layouts

By default bug.n is in tiled layout mode. Ensure it is tiled mode with:

[Win]+[t]

You should also notice the []= symbol between the tag numbers and the title bar.

bug.n has two further layout modes, floating ><> [Win]+[f] and monocle [M] [Win]+[m]. Monocle is good for ensuring a window is kept maximized and focused. This is useful in web kiosk environments and watching flash videos. Further layout modes can be included through patches.

Floating

Floating layout will be familiar to Windows users. To toggle/change a window from tile to Floating, use [Win]+[Shift]+[f]. To change the size of the floating window using keys, you have to enable resizing of the window using [Win]+[Shift]+[s] . If the starting configuration of the window, is in tile mode, pressing [Win]+[Shift]+[s], will automatically convert it into a floating window, and you can use the arrow keys to modify the window size. To move the window in floating mode, using [Win]+[Shift]+[m].

Quitting

To quit bug.n cleanly:

[Win]+[Cntrl]+[q]

Status

By default bug.n is showing bug.n in its statusbar.
